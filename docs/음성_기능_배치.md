# 음성 기능 배치 (Memorial / 하드웨어 재생)

## 배치 위치

- **Memorial 플로우**: **테마 선택 화면 (Theme Selection)** 에서 음성 녹음을 넣었습니다.
- 테마 그리드 **위쪽**, "Preview" 버튼 **아래**에 **「음성 메시지 녹음 (선택)」** 접이식 섹션이 있습니다.

## 사용자 플로우

1. 테마 선택 화면에서 테마 선택.
2. **(선택)** 「음성 메시지 녹음 (선택)」을 펼치고 **녹음** → 재생/삭제 후 유지.
3. **Preview** 클릭 시  
   - 녹음이 있으면: `mixAudioFiles(voiceBlob)` 후 base64로 **`localStorage.eternal_beam_mixed_audio`** 에 저장.  
   - 녹음이 없으면: 해당 키 제거.
4. 이후 미리보기 → NFC/콘텐츠 저장 시, 기존 로직대로 **`mixed_audio_url`** 로 서버에 전달되면 하드웨어(블루투스)에서 재생됩니다.

## 하드웨어 연동

- 앱에서 저장한 `mixed_audio`(base64) 또는 서버의 `mixed_audio_url` 은 기존 **블루투스 → 하드웨어 재생** 로직에서 사용됩니다.
- Memorial 쪽은 **테마 선택 단계에서만** 녹음/저장을 담당하고, 재생 경로는 기존 Eternal Beam 앱과 동일한 **get_playback_by_slot / mixed_audio_url** 흐름을 따릅니다.

## 구현 파일

| 역할 | 파일 |
|------|------|
| 테마 선택 + 음성 UI | `src/components/memorial/theme-selection-screen.tsx` |
| 믹싱(음성만 또는 음성+BGM) | `src/app/services/audioMixer.ts` |
| 저장 키 | `localStorage.eternal_beam_mixed_audio` (base64) |
